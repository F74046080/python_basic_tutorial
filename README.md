Through this tutorial, you will learn: 
1. 物件 － Object in python
2. 變數 － What's variable
3. 數值 － Numbers(integer and floating number)
4. 運算元、運算子 － Operator and Operand
5. 資料型態 － Common data types(boolean、string、list....)
6. 輸入、輸出 － IO
7. 條件句 － Conditional
8. 迴圈 － Loop
9. 註解 － Comment
